# Personal Portfolio Website - Week 1

## 📌 Project Overview
This project is a personal portfolio website built using HTML5 and CSS3 as part of the DecodeLabs Frontend Development Program (Week 1).

The website introduces me, showcases my skills, projects, and provides my contact information through a clean and modern user interface.

## 🚀 Features
- Responsive Header with Navigation
- Hero Section
- About Me Section
- Projects Section
- Skills Section
- Contact Section
- Footer
- Clean and Semantic HTML Structure
- Modern CSS Styling

## 🛠️ Technologies Used
- HTML5
- CSS3

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/DecodeLabs-Week-1.git
```

2. Open the project folder.

3. Open `index.html` in any web browser.

Or use **Live Server** in VS Code.
