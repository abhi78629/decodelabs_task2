# Responsive Portfolio Website - Week 2

## 📌 Project Overview

This project is the responsive version of my personal portfolio website created during Week 2 of the DecodeLabs Frontend Development Program.

The main objective was to convert the static portfolio into a fully responsive website using modern CSS techniques.

## 🚀 Features

- Responsive Navigation Bar
- Mobile-First Design
- CSS Media Queries
- Responsive Typography
- Flexbox Layout
- CSS Grid Layout
- Responsive Hero Section
- Responsive Project Cards
- Responsive Skills Section
- Optimized for Mobile, Tablet and Desktop

## 🛠️ Technologies Used

- HTML5
- CSS3
- CSS Flexbox
- CSS Grid
- Media Queries

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/DecodeLabs-Week-2.git
```

2. Open the project folder.

3. Open `index.html` in any web browser.

Or run using **Live Server** in VS Code.
